# AquesTalk-proxy

32bit バイナリでしか動かなかった旧ライセンス版 AquesTalk を Socket 越しに呼び出せるようにするプログラム。

AquesTalkのライセンス変更については[公式ブログ][blog.a-quest]を参照してください。
